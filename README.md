# 자동차 경주 게임

## 기능 요구사항

### 1,2단계. 자동차 움직임 및 우승자 결정(Car, Cars)
- [x] 자동차는 이름을 가진다.
- [x] 자동차 생성 시 이름이 5자를 초과하면 `IllegalArgumentException`을 발생시킨다.
- [x] 자동차는 무작위 값이 4 이상일 경우 전진한다.
- [x] 자동차는 무작위 값이 3 이하일 경우 정지한다.
- [x] 게임 완료 후 가장 멀리 간 자동차(들)를 우승자로 판별한다.

### 3단계. 게임 실행 (Controller & UI)
- [ ] 사용자에게 경주할 자동차 이름들을 입력받는다. (쉼표 `,` 기준으로 구분)
- [ ] 사용자에게 시도할 이동 횟수를 입력받는다.
- [ ] 0~9 사이의 무작위 값을 생성하여 각 자동차의 전진 조건으로 사용한다.
- [ ] 매 시도마다 자동차들의 이름과 현재 위치(전진 횟수만큼 `-` 출력)를 출력한다.
- [ ] 게임 종료 후 최종 우승자를 출력한다. (공동 우승 시 쉼표로 구분)

## 프로그래밍 요구사항 (제약 사항)
- **메인 메서드를 추가하여 실행 가능한 애플리케이션으로 만든다.**
- 값을 입력받기 위해 `Scanner`를, 랜덤 값 생성을 위해 `java.util.Random`을 사용한다.
- Java Style Guide를 준수한다.
- **indent(들여쓰기) depth는 1까지만 허용한다.** (메서드 분리를 적극 활용)
- **3항 연산자를 쓰지 않는다.**
- **else 예약어(switch 포함)를 쓰지 않는다.** - **메서드 길이는 15라인을 넘어가지 않도록 구현한다.** ```
