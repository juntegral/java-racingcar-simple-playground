# 자동차 경주 게임

## 단계별 기능 요구사항

### 1단계. 자동차
- [x] 자동차는 이름을 가진다.
- [x] 자동차 생성 시 이름이 5자를 초과하면 `IllegalArgumentException`을 발생시킨다.
- [x] 자동차는 무작위 값이 4 이상일 경우 전진한다.
- [x] 자동차는 무작위 값이 3 이하일 경우 정지한다.

### 2단계. 자동차 무리
- [ ] N대의 자동차가 참여하는 경주가 있다.
- [ ] 주어진 횟수 동안 N대의 자동차는 전진 또는 멈출 수 있다.
- [ ] 게임 완료 후 우승자를 판별할 수 있다.
- [ ] 우승자는 한 명 이상일 수 있다.

## 🛠️ 프로그래밍 요구사항 (제약 사항)
- 메인 메서드를 만들지 않고 테스트 코드로만 검증한다.
- Java Style Guide를 준수한다.
- **indent(들여쓰기) depth는 1까지만 허용한다.** (메서드 분리를 적극 활용)
- **3항 연산자를 쓰지 않는다.**
- **else 예약어를 쓰지 않는다.** (switch/case 포함 금지. early return 패턴 활용)
- **메서드 길이는 15라인을 넘어가지 않도록 구현한다.** (한 가지 일만 하도록 원칙 준수)
